Oracle Database Standard Edition 2 (On-Premises)

O Oracle Database Standard Edition 2 é um banco de dados acessível e completo que oferece facilidade de uso, potência e desempenho sem precedentes para aplicações de grupos de trabalho, departamentos e web. De ambientes de servidor único para pequenas empresas a ambientes de filiais altamente distribuídas, o Oracle Database Standard Edition 2 inclui todos os recursos necessários para a criação de aplicações críticas para os negócios, com suporte para clustering de serviços com o Oracle Real Application Clusters (Oracle RAC). Ele também permite que os usuários utilizem uma arquitetura multilocatário para fornecer um gerenciamento mais flexível e responsivo para seus bancos de dados no futuro.
