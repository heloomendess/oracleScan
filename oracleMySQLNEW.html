MySQL Enterprise Edition Subscription

O MySQL é o sistema de gerenciamento de banco de dados de código aberto mais conhecido no mundo. Os bancos de dados são os repositórios de dados essenciais para todas as aplicações de software. Por exemplo, sempre que alguém realiza uma pesquisa na Web, faz login em uma conta ou conclui uma transação, um banco de dados armazena as informações para que elas possam ser acessadas no futuro. O MySQL se destaca nessa tarefa.

MySQL Enterprise Edition Subscription (On-Premises, 1-4 socket server)

O MySQL Enterprise Edition inclui o conjunto mais completo de ferramentas de produção, backup, monitoramento, modelagem, desenvolvimento e administração do MySQL para que as empresas alcancem os mais altos níveis de confiabilidade, segurança e disponibilidade. Além do suporte global líder do setor da Oracle, a assinatura do MySQL Enterprise Edition reduz os riscos, custos e tempo necessários para desenvolver, implantar e gerenciar aplicativos MySQL críticos para os negócios.
