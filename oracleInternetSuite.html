Ferramenta para Desenvolvedores

A Oracle oferece um conjunto completo e integrado de ferramentas de desenvolvimento de aplicações e inteligência de negócios que suportam qualquer abordagem de desenvolvimento, qualquer plataforma tecnológica e qualquer sistema operacional. Os desenvolvedores utilizam essas ferramentas para criar aplicações sofisticadas que proporcionam maior usabilidade, visibilidade e capacidade de gerenciamento dos dados que impulsionam a solução. De Java a SQL, passando por todas as estruturas, linguagens e plataformas mais populares que impulsionam aplicações e arquiteturas de nuvem modernas, use este índice para localizar rapidamente as informações que você precisa.

Oracle Internet Developer Suite (On-Premises)

O Oracle Internet Developer Suite (Oracle iDS) é o ambiente de desenvolvimento mais completo e integrado que combina o poder das ferramentas de desenvolvimento de aplicações e de business intelligence em um único pacote, baseado nos padrões mais recentes do setor. O Oracle iDS permite que os desenvolvedores criem rapidamente aplicações transacionais de alta qualidade, que podem ser implantadas em múltiplos canais – incluindo portais, serviços web e redes sem fio – e que podem ser expandidas com recursos de business intelligence – incluindo consultas e análises ad hoc, relatórios web de alta qualidade e análises avançadas. O Oracle Internet Developer Suite inclui:

  • Desenvolvimento de aplicativos: Oracle Designer, Oracle Forms Developer, Oracle JDeveloper (incluindo Business Intelligence Beans e Business Components para Java), Oracle Software Configuration Manager.
  • Para Business Intelligence: Oracle Reports Developer, Oracle Discoverer Administrator, Oracle Warehouse Builder, Oracle Clickstream Intelligence Builder.
